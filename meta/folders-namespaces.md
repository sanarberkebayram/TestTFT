# Folders & Namespaces

Structure (under `Assets/_Project/Scripts/`):
- Runtime/ Player, Enemies, Systems, UI, Tools
- Data/ Items, Abilities, Enemies, Configs
- Editor/ Inspectors, Windows, Tools

Namespaces mirror folders (use `ProjectName` if unknown):
- `ProjectName.Scripts.Runtime.Player`
- `ProjectName.Scripts.Runtime.Systems.Inventory`
- `ProjectName.Scripts.Data.Items`
- `ProjectName.Scripts.Editor.Inspectors`
